# 第三章：几何作图

## 核心思想
仅用直尺（无刻度）和圆规探索几何可能性。这看似简单的工具，揭示了代数与几何的深层联系——很多看似几何的问题，本质是代数问题。

## 关键概念

### 尺规作图基础

**七种基本作图**：
1. 过两点作直线
2. 确定两直线交点
3. 过一点作已知直线的垂线
4. 作已知线段的垂直平分线
5. 以某点为圆心、某长度为半径作圆
6. 确定直线与圆的交点
7. 确定两圆的交点

**通俗例子**：只用这些基本操作，你能三等分一条已知线段吗？
- 可以！先画一个直角，然后等分斜边对应的弦。

**代数判据**：能够用尺规作图的线段长度，必须能从单位长度1出发，经过有限次加、减、乘、除、平方根运算得到。

**通俗理解**：每次用圆规画圆、直尺画线，在代数上对应什么？
- 交点 = 解一次或二次方程
- 每次新产生的长度 = 已有长度的 +、-、×、÷、√
- 所以能作的数必须是"二次扩域"中的数

### 三大古典问题

#### 1. 三等分角
**问题**：任意给定一个角，能否只用直尺圆规把它分成三个相等的小角？

**答案**：一般角不能三等分！

**通俗解释**：60度角可以三等分（20度），但120度角不能！原因是三等分角对应解三次方程，而尺规作图只能解二次方程。

**具体例子**：
- 三等分60° → 得到20°，可以作
- 三等分60°的思路：把角三等分相当于 cos(θ/3) 能用 cosθ 表示，但 cos(θ/3) 涉及三次方程

#### 2. 倍立方
**问题**：已知一个立方体体积，能否尺规作图出体积是两倍的立方体？

**答案**：不能！

**通俗解释**：原立方体边长1，体积1。要体积2，边长需要∛2。∛2是三次根，不是二次根，所以不能尺规作图。

**通俗故事**：古希腊人问德尔斐神庙"如何使我的国家人口翻倍"，神说"把祭坛体积加倍"。结果发现这是不可能用尺规完成的——神在开玩笑！

#### 3. 化圆为方
**问题**：能否尺规作图出一个正方形，使其面积等于已知圆的面积？

**答案**：不能！

**通俗解释**：若圆半径为1，面积=π。需要正方形边长=√π。但1882年林德曼证明了π是超越数——不是任何整系数多项式的根。因此√π也不可能用尺规作图。

**通俗类比**：想象用直尺画一条"完美光滑"的曲线，但直尺只能画直线——化圆为方就像试图用直线"完美"代替曲线，注定失败。

### 正多边形作图

**高斯的发现**：正十七边形（17边）可以用尺规作图！

**发现过程**：1796年，19岁的高斯研究多项式 x¹⁶+x¹⁵+...+x+1=0，发现其解能用根式表示的关键在于：17是费马素数。

**正n边形可作图条件**：
n = 2ᵏ × p₁ × p₂ × ... × pₘ
其中每个pᵢ都是**不同的费马素数**（形如 2^(2^t) + 1 的素数）

**费马素数**：已知的有 F₀=3, F₁=5, F₂=17, F₃=257, F₄=65537
```
正3边形（三角形）可作：3 = 3 ✓
正5边形可作：5 = 5 ✓
正6边形可作：6 = 2×3 ✓
正7边形不可作：7是普通素数 ✗
正17边形可作：17 = 17 ✓
正255边形不可作：255 = 3×5×17，但17重复了 ✗
```

### 域扩张与作图

**通俗理解二次扩域链**：
- 初始：只有有理数（能做出的最简单的数）
- 第一次扩张：加入√2，得到 {a + b√2 | a,b∈Q}
- 第二次扩张：再加入√3，得到 {a + b√2 + c√3 + d√6 | a,b,c,d∈Q}
- 每次扩张对应一次作图操作

**例子**：已知单位长度1和长度2，能否作长度√3？
1. 画长为2的线段AB
2. 取其中点O，以O为圆心、1为半径画半圆
3. 从A做垂线交半圆于C
4. AC = √3 ✓

## 重要定理

### 高斯-旺策尔定理
正n边形可用尺规作图 ⟺ n = 2ᵏ × 费马素数（各费马素数仅出现一次）

### 费马数
Fₖ = 2^(2ᵏ) + 1
```
F₀ = 2¹ + 1 = 3（素数）
F₁ = 2² + 1 = 5（素数）
F₂ = 2⁴ + 1 = 17（素数）
F₃ = 2⁸ + 1 = 257（素数）
F₄ = 2¹⁶ + 1 = 65537（素数）
F₅ = 2³² + 1 = 4294967297 = 641×6700417（合数！）
```

## 思维启示
几何作图的限制揭示了**代数结构决定几何可能性**这一深刻原理。看似"几何"的问题（三大古典难题），本质上是不了解代数域扩张——一旦看清代数本质，几何可能性就一目了然。
