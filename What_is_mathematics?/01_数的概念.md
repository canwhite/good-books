# 第一章：数的概念

## 核心思想
数学的起点是**自然数**（1, 2, 3, ...），由此逐步扩展到整数、有理数、无理数和复数。每一次扩展都解决了前一个数系的缺陷。

## 关键概念

### 自然数与整数

**皮亚诺公理**用五条公理定义了自然数：
1. 1是自然数
2. 每个自然数都有后继数
3. 1不是任何自然数的后继
4. 不同自然数有不同的后继
5. 若某性质对1成立，且对后继成立，则对所有自然数成立（数学归纳法原理）

**数学归纳法**的例子：证明"从1加到n等于 n(n+1)/2"
- 验证n=1时成立：1 = 1×2/2 = 1 ✓
- 假设对k成立，证明对k+1成立
- 1+2+...+k+(k+1) = k(k+1)/2 + (k+1) = (k+1)(k+2)/2 ✓

**通俗例子**：数羊。牧羊人不需要"0"只羊——"0"是后来才发明出来的！

### 有理数

**有理数稠密性**：任意两有理数之间必有另一有理数。

**通俗例子**：1和2之间有无数个分数——1.1, 1.01, 1.001... 而且任意两个分数之间还能再插分数，像有理数这样"密密麻麻"。

**十进制小数视角**：
- 1/4 = 0.25（有限小数）
- 1/3 = 0.333...（无限循环小数）
- 所有有理数都是**有限或循环小数**

### 无理数

**什么是无理数**：不是有理数的实数，即不能表示为两个整数之比 a/b（b≠0）的数。

**无理数的小数特征**：无理数的十进制小数是**不循环的无限小数**。

**√2无理性的经典证明**（反证法）：
假设√2 = a/b（a,b互质），则 a² = 2b²
→ a²是偶数 → a是偶数，设a=2c
→ 4c² = 2b² → 2c² = b²
→ b²是偶数 → b是偶数
→ a和b都是偶数，与"互质"矛盾！
∴ √2不能写成两个整数之比

**通俗例子**：你画一个边长为1的正方形，对角线长度就是√2。这个长度不能表示成分数，它"不循环保守秘密"。

**连分数例子**：黄金比例 φ = (1+√5)/2 ≈ 1.618
```
φ = 1 + 1/(1 + 1/(1 + 1/(1 + ...)))
```
越往后越接近φ，是最美的无理数表示。

**超越数**：不是任何整系数代数方程的根。e和π都是超越数——这意味着你永远不能用直尺和圆规"化圆为方"。

### 实数

**实数 = 有理数 + 无理数**

实数填补了数轴上所有"空隙"，每一个实数都对应数轴上的一个点，反之亦然。

**实数的阿基米德性**：
对任意正实数a、b，存在整数n使得 na > b。

**通俗例子**：不管一根棍子多短（a），只要量足够多次（n），就一定能超过一根长绳子（b）。这就是"滴水穿石"的数学原理——只要你重复得够多。

**实数的小数表示**：
- 有理数：有限小数 或 无限循环小数
- 无理数：无限不循环小数

**例子**：
```
3.1415926... = π（不循环，超越数）
1.4142135... = √2（不循环，代数数）
0.33333... = 1/3（循环）
```

**实数的四则运算**：
实数对加、减、乘、除（除数不为0）封闭——运算结果仍是实数。

**实数与数轴的对应**：
- 每个实数 = 数轴上一个唯一的点
- 每个数轴上的点 = 一个实数
- 这是"完备性"的直观体现：没有"空隙"

**实数的有序性**：
实数可以比较大小。任意两个实数a<b，必存在c使得 a<c<b。

**通俗例子**：无论两个实数多接近，比如3.1415926和3.1415927，它们之间还能塞进无穷多个实数——这是实数"稠密性"的体现。

### 实数完备性

**戴德金分割**：把有理数分成"小于√2的"和"大于等于√2的"两组，这个分割本身就定义了一个无理数。

**通俗例子**：数轴上密密麻麻，但有理数"有空隙"——戴德金分割就是用来填这些空隙的。比如√2、π这些无理数就是用分割定义的。

**确界原理**：任何有上界的集合必有最小上界（确界）。

**通俗例子**：你把一本书放在桌上，书上放第二本，再放第三本……每放一本就记下最上层的高度。这个"高度记录"有上界（比如天花板），所以必有最小上界——这就是确界。

### 复数

**i² = -1**：引入一个"幻想"的数，让方程 x² + 1 = 0 有解。

**通俗例子**：负数的平方根本来"不存在"，但数学家说"假设它存在"，于是发明了i。这就像问"如果你有100元，还欠人50元，你现在有多少钱？"答案是50，但如果你欠人150元，答案是-50。复数让我们可以讨论更抽象的"维度"。

**复平面几何表示**：
- 实数是横轴
- 纯虚数是纵轴
- 复数 a+bi 对应平面上的点(a, b)

**代数基本定理**：任何n次多项式恰有n个复数根（重根按重数计）。这意味着复数域是"完备"的——你不需要再扩展数系了。

## 经典定理
1. **√2是无理数**——古希腊人发现这个结论时据说非常震惊
2. **素数无穷多**——欧几里得的优雅证明：假设有限个，构造新素数
3. **任何有理数是有限或循环小数**
4. **e和π是超越数**——林德曼1882年证明

## 思维启示
数的概念扩展史揭示了数学思想的本质：**不断推广抽象，满足实际与理论的双重需求**。就像乐高积木，每次扩展都打开了新的可能世界。
