# 第四章：代数

## 核心思想
代数是研究运算规则和方程求解的学科。真正的代数不是"解方程"，而是理解**结构**——群、环、域这些抽象结构才是代数的核心。

## 关键概念

### 多项式与方程

**多项式除法例子**：计算 (x³ - 6x² + 11x - 6) ÷ (x - 2)
```
x³ - 6x² + 11x - 6 = (x-2)(x²-4x+3) + 0
验证：(x-2)(x²-4x+3) = x³-4x²+3x-2x²+8x-6 = x³-6x²+11x-6 ✓
```

**因式定理**：若f(a)=0，则(x-a)是f(x)的因式
**例子**：f(1)=1-3+2=0，所以(x-1)是因式

**有理根定理例子**：找 x³ - 4x² + x + 6 = 0 的有理根
- 可能的分子：±1, ±2, ±3, ±6
- 可能的分母：1
- 所以可能的有理根：±1, ±2, ±3, ±6
- f(1) = 1-4+1+6 = 4 ≠ 0
- f(2) = 8-16+2+6 = 0 → x=2是根！
- 因式分解：(x-2)(x²-2x-3) = (x-2)(x-3)(x+1)
- 所以根是：2, 3, -1

**艾森斯坦判别法例子**：证明 x⁴ + 2x² + 2 在有理数域不可约
取p=2：
- 系数：1, 0, 2, 0, 2
- p整除0,2,0,2，但不整除首项系数1 ✗ 不适用...

换一个：x² + 2x + 2
- 系数：1, 2, 2
- p=2：2|2, 2|2，但2∤1（首项），且p²=4∤2（常数项）
- ✓ 在有理数域不可约

### 代数基本定理

**n次方程恰有n个复数根**

**通俗例子**：二次方程 x² - 5x + 6 = 0
- 判别式 Δ = 25 - 24 = 1 > 0，两个不同实根：x=2, x=3
- x² - 2x + 1 = 0：Δ=0，两个相等实根：x=1, x=1（重根）
- x² + 1 = 0：Δ = 1 - 4 = -3 < 0，两个共轭复根：x=i, x=-i

**几何意义**：把方程右边设为0，在复平面上画图像，曲线总会和"水平线"相交n次（重根算两次）。

### 代数扩域

**最小多项式例子**：√2 + √3 在有理数域上的最小多项式
设 x = √2 + √3
→ x - √3 = √2
→ (x - √3)² = 2
→ x² - 2x√3 + 3 = 2
→ x² + 1 = 2x√3
→ (x² + 1)² = 12x²
→ x⁴ - 10x² + 1 = 0
所以最小多项式是 x⁴ - 10x² + 1（可验证次数最小）

**尺规作图的代数解释**：为什么三等分角不行？
- 三等分角涉及 cos(θ/3)，需要解三次方程
- 但尺规作图只能解二次方程（对应二次扩域）
- 三次扩域 > 二次扩域能力范围
- ∴ 一般角不能三等分

### 伽罗瓦理论初步

**核心思想**：方程可解性 ↔ 根的对称群结构

**通俗例子**：二次方程 ax² + bx + c = 0
- 两个根 x₁, x₂
- 根的对称：交换x₁和x₂
- 这个"交换群"只有两个元素：{恒等变换, 交换}
- 这个群足够"小"，所以二次方程有根式解

**五次方程无根式解**：
- 一般的五次方程有5个根
- 根的对称群是S₅（5个元素的所有排列）
- S₅是"不可解"的群（虽然这是抽象的概念）
- 因此一般五次方程没有根式解

### 线性代数基础

**向量空间例子**：所有次数≤2的多项式构成一个向量空间
- 加法：(x²+1) + (x+2) = x²+x+3
- 数乘：3·(x²+1) = 3x²+3
- 基底：{1, x, x²}（三维空间）
- 维度：3

**行列式例子**：
```
| 1  2 |
| 3  4 | = 1×4 - 2×3 = 4 - 6 = -2
```
几何意义：两个列向量张成的平行四边形面积（带符号）

**特征值与特征向量例子**：
矩阵 A = [[2,1],[1,2]]
解 det(A-λI) = 0：
```
|2-λ  1  |
|1    2-λ| = (2-λ)² - 1 = λ² - 4λ + 3 = 0
→ λ = 1 或 λ = 3
```
- λ=1时：特征向量 [1,-1]ᵀ（方向不变）
- λ=3时：特征向量 [1,1]ᵀ（方向不变，拉伸3倍）

## 经典问题

### 五次方程无根式解
1824年，阿贝尔证明了一般五次方程不能用根式求解。

**通俗理解**：二次方程有求根公式，三次、四次也有。但到了五次，数学家们试了两百多年都没找到——因为根本不存在！

### 代数基本定理
高斯给出了四种不同证明。

**拓扑证明思路**：
- 把复数代入多项式，观察当|z|很大和很小时的行为
- 用连续性说明曲线必定经过原点
- 从而证明根的存在

## 思维启示
代数的本质是**符号操作与结构研究**。从具体方程到抽象域论，体现了数学的高度概括性——当你理解了"结构"，具体计算反而变得次要。
