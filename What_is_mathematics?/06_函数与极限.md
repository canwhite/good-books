# 第六章：函数与极限

## 核心思想
函数是数学分析的核心对象，极限是微积分的基础。理解极限的关键是：**无限接近但永远达不到**。ε-δ语言是数学严谨化的里程碑。

## 关键概念

### 实数连续性

**确界原理例子**：集合 S = {x | x² < 2}
- 有上界：比如2，因为所有元素都<2
- 有最小上界吗？有！就是√2 ≈ 1.414...
- √2本身不在S里（S里只有平方<2的数）
- 但√2是确界——S的所有元素都≤√2，且没有更小的上界

**通俗理解**：你可以无限接近√2，但S里永远不包含√2。确界原理说：这样的"空缺"有最小上界。

**区间套定理例子**：
- 第一个区间：[1, 2]
- 第二个区间：[1.4, 1.5]
- 第三个区间：[1.41, 1.42]
- ...
这些区间一个套一个，越来越窄
→ 它们必定有公共点！这个公共点就是√2

**通俗理解**：像俄罗斯套娃，一层包一层，总有一个最深处是所有层共有的那一点。

### 函数极限

**ε-δ语言的通俗理解**：

"当x无限接近a时，f(x)无限接近L"

**白话版**：无论你要求f(x)多么接近L（只要别太贪心要求绝对精确），我都能找到x的范围，让f(x)达到你的要求。

**例子**：证明 lim(x→1) (2x+1) = 3

给定ε>0，要找δ使得：|x-1|<δ → |(2x+1)-3|<ε
即 |x-1| < ε/2
所以取 δ = ε/2 即可。

**通俗说**：想让(2x+1)离3不超过0.01，只需让x离1不超过0.005。

### 连续性

**连续通俗定义**：画图像时不需要"抬笔"

**介值定理例子**：今天气温8度，昨天气温15度。介值定理说：**中间一定有一个时刻气温是12度**（或任何8和12之间的温度）。

**等电梯例子**：电梯从1楼到10楼，中间必定经过5楼。连续函数就是"电梯函数"——它不会瞬移。

**最值定理例子**：
- 一天的温度变化是连续的
- 在闭区间（比如0点到24点）上必有最高温度和最低温度
- 不会无限上升也不会无限下降

**一致连续 vs 连续**：
- **连续**：在每个点都连续（局部性质）
- **一致连续**：整个区间上连续性"均匀"（全局性质）

**通俗例子**：f(x) = x² 在[0,1]上一致连续——无论取多小，区间上所有点都能"同步"满足要求。但在[1,∞)上不是一致连续的：x越大，函数变化越快。

### 特殊函数类型

**单调函数例子**：
- f(x) = x³ 是单调递增的（每步往上走）
- 至多只有可数个间断点——相对"乖"的函数

**有界变差函数例子**：
- 股票价格走势图是有界变差的——总有个上下限
- 可以分解为两个增函数之差（盈利-亏损的视角）

**黎曼可积条件例子**：
- 连续函数必可积 ✓
- 有有限个间断点的有界函数可积 ✓
- 狄利克雷函数（在有理点=1，无理点=0）不可积 ✗（因为间断点太多）

## 经典问题

### 维尔斯特拉斯函数（处处连续处处不可导）
**通俗理解**：构造一个"极端扭曲"的函数——在每个点都"尖锐"，以至于在任何点都画不出切线。

**通俗类比**：想象一根无限皱折的海岸线，它的每一点都像悬崖，无法定义"平滑的切线方向"。数学上可以构造出这样的函数，但在自然界中很难见到。

### 康托尔集
**构造方法**：
1. 取[0,1]区间
2. 去掉中间1/3 → 剩下[0,1/3]和[2/3,1]
3. 每个区间再去掉中间1/3 → 剩下4段
4. 无限重复...

**神奇性质**：
- 总长度 = 0（挖空了）
- 但包含**无穷多**个点
- 还是**不可数**的（和实数一样"多"）
- 是一种"分形"——局部和整体相似

**通俗类比**：像沙子筛了无数遍，虽然总长度为零，但"沙粒"还在。

### 黎曼函数
```
f(x) = 1/q，若x = p/q（最简分数）
f(x) = 0，若x是无理数
```

**性质**：
- 在所有**无理点**连续（无理数附近的分数值可以任意小）
- 在所有**有理点**不连续（有理点附近有无理数，函数值为0）

**通俗理解**：这个函数"讨厌"有理数，却对无理数很友好——在无理数处它是安静的，在有理数处它是躁动的。

## 思维启示
极限的ε-δ语言是数学严谨化的里程碑，它解决了第二次数学危机（无穷小量的悖论）。理解极限的关键不是"等于"，而是"无限接近"——这个思想贯穿整个分析学。
