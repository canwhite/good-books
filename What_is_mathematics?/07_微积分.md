# 第七章：微积分

## 核心思想
微积分研究**变化**与**累积**：
- **微分**：已知整体，求局部变化率（本质是求导）
- **积分**：已知局部变化，求整体累积量（本质是累积求和）
- 两者**互为逆运算**——这是微积分基本定理的核心

**极限与微积分的关系**：
- **导数** = 极限：瞬时变化率是 Δx→0 时的极限
- **定积分** = 极限：把区间无限细分，每份累积求和的极限（如位移是速度的累积，体积是截面积的累积）

**极限与微积分的关系**：
- **导数** = 极限：瞬时变化率是 Δx→0 时的极限
- **定积分** = 极限：把区间无限细分，每份累积求和的极限（如位移是速度的累积，体积是截面积的累积）
- **微积分基本定理**：导数和积分互为逆运算——本质上都是极限

## 关键概念

### 导数

**导数的物理意义——瞬时速度**：

**通俗例子**：开车时，里程表显示你开了100公里，用了2小时。"平均速度"是50km/h。但仪表盘显示的当前速度呢？

- 某一时刻的速度 = 当时间变化趋于0时，距离/时间的极限
- 这就是导数：v(t) = lim(Δt→0) [s(t+Δt)-s(t)]/Δt = s'(t)

**导数的几何意义——切线斜率**：

**通俗例子**：山坡的陡峭程度。在山上某点画一条刚好"贴合"山坡的直线——这就是切线。切线有多陡，导数就有多大。

**基本导数公式**：
```
(d/dx) xⁿ = nxⁿ⁻¹          例：(d/dx) x³ = 3x²
(d/dx) sin x = cos x
(d/dx) eˣ = eˣ
(d/dx) ln x = 1/x
```

**链式法则例子**：
求 d/dx sin(x²) = ?
设 u = x²，则 du/dx = 2x
d/dx sin(u) = cos(u) · du/dx = cos(x²) · 2x = 2x cos(x²)

**通俗理解**：复合函数求导是"链式反应"——层层剥开，每层求导再相乘。

### 积分

**定积分的几何意义——曲边梯形面积**：

**通俗例子**：计算抛物线 y=x² 从0到1下方覆盖的面积。
- 矩形面积 = 底×高，太粗糙
- 把区间分成n个小窄条，每个窄条近似成矩形
- n越大，越接近真实面积
- 极限情况就是定积分 = 1/3

**黎曼和的概念**：
∫₀¹ x² dx = lim(n→∞) [1/n · ((1/n)² + (2/n)² + ... + (n/n)²)]
= lim(n→∞) [1/n³ · (1² + 2² + ... + n²)]
= lim(n→∞) [1/n³ · n(n+1)(2n+1)/6] = 1/3

**原函数与不定积分**：
- 求导的逆运算
- 若 F'(x) = f(x)，则 ∫f(x)dx = F(x) + C
- 例：∫x² dx = x³/3 + C

### 微积分基本定理

**连接导数与积分的核心定理**：

**通俗理解**：
- 导数 = "变化率"（已知整体累积，求瞬时变化）
- 积分 = "累积量"（已知瞬时变化，求整体累积）
- 基本定理说：**它们是互逆的！**

**例子**：
已知速度 v(t) = 3t²，求位移 s(t)
- s(t) = ∫v(t)dt = t³ + C
- 若 s(0)=0，则 s(t) = t³

反过来，已知位移 s(t) = t³，求速度
- v(t) = s'(t) = 3t² ✓

**物理直观**：开车时里程表（积分）和速度表（导数）显示的是同一辆车在不同视角下的信息。

### 泰勒展开

**泰勒公式**：
f(x) = f(a) + f'(a)(x-a) + f''(a)/2! · (x-a)² + ... + f⁽ⁿ⁾(a)/n! · (x-a)ⁿ + Rₙ

**通俗理解**：用多项式"逼近"复杂函数。多项式最简单——只有加减乘除。

**例子**：在x=0处展开 eˣ
```
eˣ = 1 + x + x²/2! + x³/3! + x⁴/4! + ...
```
取前几项：e^0.5 ≈ 1 + 0.5 + 0.125 + 0.0208 + 0.0026 ≈ 1.6484
实际 e^0.5 ≈ 1.6487，误差仅0.0003！

**通俗类比**：泰勒展开像"升级版的一次函数"。一次函数 y=a+bx 用直线逼近曲线。更高阶的展开用抛物线、立方曲线...越来越精确。

### 多元微积分初步

**偏导数例子**：
f(x,y) = x² + xy + y²
∂f/∂x = 2x + y（把y当常数，只对x求导）
∂f/∂y = x + 2y（把x当常数，只对y求导）

**通俗理解**：多变量函数中，"偏导数"只看一个方向的变化——像只调整x而固定y不动。

**梯度向量**：
∇f = (∂f/∂x, ∂f/∂y)
梯度指向函数增长最快的方向！

**通俗例子**：下山时，梯度向量指向最陡的上坡方向。沿梯度走，函数值增加最快；逆梯度走，下降最快。

## 经典定理

### 均值定理
存在c∈(a,b)使 f'(c) = [f(b)-f(a)]/(b-a)

**通俗理解**：从a到b的连续函数图像，必有一点切线斜率等于整段的平均斜率。

**例子**：平均速度50km/h，一定存在某个时刻瞬时速度正好是50km/h。

### 洛必达法则
处理 0/0 或 ∞/∞ 型不定式

**例子**：
lim(x→0) sin x / x = 0/0，用洛必达：
= lim(x→0) cos x / 1 = 1/1 = 1

**通俗理解**：分子分母都趋于0时，比值取决于它们"消失"的速度。洛必达通过求导来比较它们消失的快慢。

### 泰勒中值定理
拉格朗日余项形式：
Rₙ(x) = f⁽ⁿ⁺¹⁾(ξ)/(n+1)! · (x-a)ⁿ⁺¹（ξ在a和x之间）

**意义**：告诉我们泰勒展开的误差有多大，可以精确控制。

## 思维启示
微积分是数学史上最重要的发明之一，它统一了**静态与动态**、**局部与整体**的数学描述。牛顿和莱布尼茨各自独立发现微积分，开启了现代数学的大门。
